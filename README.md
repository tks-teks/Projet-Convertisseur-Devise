# Projet-Convertisseur-Devise

Ce projet représente le fruit des recherches des étudiants de Keyce Informatique Bachelor2 Groupe 6. Son objectif est de créer un convertisseur de devise moderne et rapide, développé avec le framework React.js.

## 🚀 Fonctionnalités
✅ Conversion instantanée entre plusieurs devises 📊

✅ Interface simple et intuitive 💡

✅ Données mises à jour grâce à une API de taux de change 🌍
